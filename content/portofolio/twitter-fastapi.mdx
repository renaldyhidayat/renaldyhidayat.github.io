# Twitter Clone FastAPI

Proyek ini adalah sebuah kloning dari Twitter yang dibuat menggunakan FastAPI. Framework ini dipilih karena kemiripannya dengan Express.js, serta kecepatan dan efisiensinya dalam membangun REST API.

## Features

- CRUD postingan
- Autentikasi JWT
- Realtime WebSocket untuk notifikasi

## Technologies Used

- FastAPI
- WebSockets
- PostgreSQL
