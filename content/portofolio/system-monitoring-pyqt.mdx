# 🖥️ System Monitoring Dashboard

**System Monitoring with PyQt** is a real-time monitoring application that provides insights into system performance, including CPU usage, RAM utilization, network activity, and hardware statistics. Built with PyQt and PyQtGraph, this dashboard presents data visually, making it easy to track system health.

## ✨ Features

🚀 **Real-time CPU & RAM Monitoring** - Displays usage trends over time.

📡 **Network Activity Tracking** - Monitors upload and download speeds dynamically.

🔥 **CPU Temperature & Frequency** - Visualizes thermal and performance metrics.

🔋 **Battery Status** - Provides information on power status and remaining battery life.

📊 **Live Graphs with PyQtGraph** - Smooth and interactive visualization of system metrics.
