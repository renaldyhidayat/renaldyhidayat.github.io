# Ecommerce FastAPI React

Proyek ini adalah sebuah e-commerce yang dibuat menggunakan FastAPI di backend dan React di frontend. FastAPI dipilih karena performanya yang cepat serta memiliki sintaks mirip dengan Express.js.

## Features

- API berbasis FastAPI
- Frontend dengan React dan Tailwind CSS
- Sistem autentikasi JWT

## Technologies Used

- FastAPI
- React.js
- PostgreSQL
