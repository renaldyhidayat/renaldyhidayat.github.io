## Deskripsi Proyek

Proyek ini bertujuan untuk mengembangkan sebuah sistem rekomendasi produk untuk platform e-commerce dengan menggunakan analisis sentimen dari ulasan pengguna dan peringkat produk. Sistem ini membantu pelanggan untuk menemukan produk yang paling relevan berdasarkan pandangan dan pengalaman pengguna lain.

### Fitur Utama
- **Analisis Sentimen:** Menggunakan *Natural Language Toolkit* (NLTK) untuk melakukan analisis sentimen terhadap ulasan pengguna, mengategorikan mereka sebagai positif, netral, atau negatif.
- **Rekomendasi Produk:** Menyediakan rekomendasi produk yang disarankan berdasarkan ulasan pengguna dan rating produk.
- **Pengolahan Data:** Menggunakan Pandas untuk pengolahan data ulasan dan rating sebelum diteruskan untuk analisis lebih lanjut.

### Teknologi yang Digunakan
- **Backend:** Django untuk server dan API.
- **Frontend:** React JS dengan Bootstrap 5 untuk tampilan antarmuka pengguna yang responsif.
- **Analisis Sentimen:** NLTK untuk analisis ulasan dan rating produk.
- **Data Processing:** Pandas untuk pengolahan data ulasan dan rating.
- **Desain Desktop:** Tauri JS untuk membangun aplikasi desktop ringan yang berbasis pada teknologi web.
