## Deskripsi Proyek

Proyek ini adalah eksperimen dalam membangun platform e-commerce dengan berbagai teknologi modern. Sistem ini mengintegrasikan berbagai layanan penting seperti payment gateway Midtrans dan Raja Ongkir untuk pengiriman, serta memiliki frontend yang dirancang untuk kemudahan pengguna dan manajemen dengan React.js dan Tauri.

### Fitur Utama
- **Backend:** Menggunakan **Nest.js** dan **PostgreSQL** untuk menyimpan data transaksi dan produk, serta mengintegrasikan dengan Midtrans untuk pembayaran dan Raja Ongkir untuk layanan pengiriman.
- **Frontend:** **React.js** dan **Redux Toolkit** digunakan untuk membangun antarmuka pengguna, dengan **Tailwind** dan **Flowbite** untuk desain CSS yang responsif.
- **Frontend Admin:** Aplikasi desktop untuk admin menggunakan **Tauri** dan **Rust**, dengan **React.js** dan **Redux Toolkit** untuk manajemen state dan **Bootstrap** dan **Mazer Dashboard** untuk tampilan admin.

### Teknologi yang Digunakan
- **Backend:** Typescript, Nest.js, PostgreSQL
- **Payment Gateway:** Midtrans
- **Pengiriman:** Raja Ongkir
- **Frontend:** Javascript, React.js, Redux Toolkit, Tailwind, Flowbite
- **Frontend Admin:** Tauri, Rust, React.js, Redux Toolkit, Bootstrap, Mazer Dashboard
