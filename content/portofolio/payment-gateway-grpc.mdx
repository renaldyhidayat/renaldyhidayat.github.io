# **Welcome to the Payment Gateway Project!**

This system simulates a payment gateway handling operations such as:

- Card Transactions 💳
- Top-ups 💰
- Withdrawals 🏧
- Transfers 🔄

The project involves multiple entities like users, cards, merchants, and financial transactions.

## **Tech Stack** 🚀

- **Golang** 🐹
- **Sqlc** 🗄️
- **Echo Framework** ⚡
- **gRPC** 🔌
