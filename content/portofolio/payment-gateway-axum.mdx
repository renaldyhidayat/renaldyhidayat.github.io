## Tech Stack

- Rust
- SeaORM
- Axum
- Docker Compose



## How the Payment Gateway Works (Using Axum + SeaORM)

<img src="https://github.com/MamangRust/payment-gateway-axum/raw/main/payment_gateway.png" alt="payment" />

### Migrate

```sh
sea-orm-cli migrate init
```

### Generate Entity

```sh
sea-orm-cli generate entity -o src/entities
```

### Migrate run UP

```sh
sea-orm-cli migrate up
```

### Migrate run Down

```sh
sea-orm-cli migrate down
```
