## Deskripsi Proyek

Proyek **Movie GRPC** adalah sebuah aplikasi **CRUD** untuk manajemen film yang dibangun dengan **Golang dan GORM**. Aplikasi ini menggunakan **gRPC** sebagai komunikasi antar layanan, serta didukung oleh observabilitas menggunakan **Jaeger, OpenTelemetry (Otel), dan Prometheus** untuk monitoring dan tracing. Seluruh layanan berjalan dalam lingkungan **Docker** untuk kemudahan deployment.

### Fitur Utama
- **CRUD Film:** Menggunakan **GORM** sebagai ORM untuk database.
- **Komunikasi gRPC:** API dikembangkan menggunakan **gRPC** untuk efisiensi dan performa tinggi.
- **Observabilitas:** Implementasi **Jaeger dan OpenTelemetry (Otel)** untuk distributed tracing.
- **Monitoring:** Integrasi dengan **Prometheus** untuk pemantauan metrik sistem.
- **Containerized:** Semua layanan berjalan dalam **Docker** untuk kemudahan pengelolaan dan deployment.

### Teknologi yang Digunakan
- **Backend:** Golang, Gin, gRPC
- **Database:** GORM (ORM untuk Golang)
- **Observabilitas:** Jaeger, OpenTelemetry (Otel)
- **Monitoring:** Prometheus
- **Containerization:** Docker
