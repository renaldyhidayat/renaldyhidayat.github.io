# Inventory Django

Project ini dibuat menggunakan Django sebagai framework utama. Django dipilih karena kesederhanaannya serta fitur yang lengkap untuk pengelolaan database dan routing.

## Features

- CRUD data dengan Django ORM
- Autentikasi pengguna
- Dashboard interaktif

## Technologies Used

- Django
- PostgreSQL
- Bootstrap
