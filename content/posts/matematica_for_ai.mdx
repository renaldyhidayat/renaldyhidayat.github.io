## Basic Mathematical Operations in Python

Python is a versatile programming language that allows you to perform a wide range of mathematical operations. In this tutorial, we'll explore some of the fundamental mathematical operations in Python.

## Addition

You can perform addition in Python using the `+` operator:

```python
# Addition
result = 5 + 3
print("5 + 3 =", result)
```
