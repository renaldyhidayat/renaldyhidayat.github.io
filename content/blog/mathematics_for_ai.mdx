# Matematika for Artificial Intelligence

Matematika adalah bahasa yang mendasari Kecerdasan Buatan (AI) dan Machine Learning (ML). Dalam artikel ini, kita akan menjelaskan konsep matematika dasar yang digunakan dalam AI, termasuk Aljabar Linear, Kalkulus, dan Statistik, serta memberikan contoh penggunaannya.

# Aljabar Linear

Dalam matematika, Aljabar Linear adalah cabang yang penting dan digunakan luas dalam berbagai disiplin ilmu, termasuk Kecerdasan Buatan. Mari kita jelajahi konsep Aljabar Linear dari tingkatan kesulitan yang berbeda.

## Aljabar linier 1 Dimensi

In one dimension, linear algebra operations typically involve scalar values represented as real numbers. Here are some mathematical representations:

Scalar Addition:

```
Scalar1 + Scalar2 = 3 + 5 = 8
```

Scalar Multiplication:

```
Scalar * Scalar1 = 2 * 3 = 6
```

```python
import numpy as np


scalar1 = 3
scalar2= 5

result_addition = scalar1 + scalar2


print("Scalar 1 + Scalar 2 = ", result_addition)


scalar = 2

result_multiplication = scalar * scalar1
print("Scalar * Scalar 1 = ", result_multiplication)

```

## Aljabar linier 2 Dimensi

In two dimensions, you work with vectors represented as pairs of numbers (x, y). Here are some mathematical representations:

Vector Addition:

```
Vector1 + Vector2 = (2, 3) + (4, 5) = (2 + 4, 3 + 5) = (6, 8)

```

Scalar Multiplication:

```
Scalar * Vector1 = 2 * (2, 3) = (2 * 2, 2 * 3) = (4, 6)
```

Dot Product:

```
Dot Product of Vector1 and Vector2 = (2, 3) · (4, 5) = 2 * 4 + 3 * 5 = 8 + 15 = 23
```

```python

vector1 = np.array([2, 3])
vector2 = np.array([4, 5])


result_addition = vector1 + vector2
print("Vector 1 + Vector 2 = ", result_addition)


scalar = 2
result_multiplication = scalar * vector1
print("Scalar * Vector 1 = ", result_multiplication)


dot_product = np.dot(vector1, vector2)
print("Dot Product of Vector1 and Vector 2 = ",  dot_product)
```

## Aljabar linier 3 Dimensi

In three dimensions, vectors are represented as triplets (x, y, z). Here are some mathematical representations:

Vector Addition:

```
Vector1 + Vector2 = (2, 3, 1) + (4, 5, 2) = (2 + 4, 3 + 5, 1 + 2) = (6, 8, 3)
```

Scalar Multiplication:

```
Scalar * Vector1 = 2 * (2, 3, 1) = (2 * 2, 2 * 3, 2 * 1) = (4, 6, 2)
```

Cross Product (in three dimensions):

```
Cross Product of Vector1 and Vector2 = (2, 3, 1) × (4, 5, 2) = ((3 * 2 - 5 * 1), (2 * 4 - 1 * 2), (4 * 5 - 3 * 2)) = (1, 6, 14)

```

# Calculus

Kalkulus adalah cabang matematika yang mempelajari perubahan dan laju perubahan. Dalam materi ini, kami akan menjelaskan tiga konsep dasar Kalkulus: Turunan, Integral Tak Tentu, dan Integral Tentu, dan juga menunjukkan bagaimana cara menggunakan Python untuk menghitungnya.

## Konsep 1: Turunan

**Definisi:** Turunan mengukur laju perubahan fungsi terhadap perubahan variabel independen. Turunan fungsi \( f(x) \) dilambangkan sebagai \( f'(x) \) atau \(\frac{}{}\).

**Contoh Penggunaan Python:**

Untuk menghitung turunan suatu fungsi, kita dapat menggunakan pustaka `SymPy` di Python. Mari kita contohkan dengan fungsi \( f(x) = 2x^2 + 3x + 1 \).

```python
# Mengimpor pustaka SymPy
import sympy as sp

# Membuat simbol variabel
x = sp.symbols('x')

# Menentukan fungsi
f = 2*x**2 + 3*x + 1

# Menghitung turunan pertama
f_prime = sp.diff(f, x)

# Menampilkan hasil
print("Turunan pertama dari f(x) adalah:", f_prime)

```

## Konsep 2: Integral Tak Tentu

Integral tak tentu adalah salah satu konsep penting dalam kalkulus. Integral ini merupakan operasi kebalikan dari turunan. Integral tak tentu dari sebuah fungsi \(f(x)\) dilambangkan sebagai \(\int f(x) \, dx\) dan menghasilkan fungsi lain yang, jika di-derivatifkan, akan menghasilkan \(f(x)\) kembali.

**Contoh Konsep Integral Tak Tentu:**

Misalkan kita memiliki fungsi \(f(x) = 3x^2 + 2x + 4\). Untuk menghitung integral tak tentu dari \(f(x)\), kita dapat menulisnya sebagai berikut:

\[
\int (3x^2 + 2x + 4) \, dx
\]

Kemudian, kita dapat menghitung integral tak tentu ini menggunakan Python dan pustaka SymPy:

```python
# Mengimpor pustaka SymPy
import sympy as sp

# Membuat simbol variabel
x = sp.symbols('x')

# Menentukan fungsi
f = 3*x**2 + 2*x + 4

# Menghitung integral tak tentu
integral = sp.integrate(f, x)

# Menampilkan hasil
print("Integral tak tentu dari f(x) adalah:", integral)
```

## Konsep 3: Integral Tentu

Integral tentu adalah salah satu konsep penting dalam kalkulus yang digunakan untuk menghitung luas di bawah kurva fungsi antara dua batas tertentu. Integral tentu dari sebuah fungsi \(f(x)\) dalam interval \([a, b]\) dilambangkan sebagai \(\int\_{}^{} f(x) \, dx\) dan menghasilkan nilai numerik yang mewakili luas daerah di bawah kurva \(f(x)\) antara \(x = a\) dan \(x = b\).

**Contoh Konsep Integral Tentu:**

Misalkan kita ingin menghitung integral tentu dari fungsi \(g(x) = 2x\) dalam interval \([1, 5]\):

\[
\int\_{1}^{5} 2x \, dx
\]

Dengan menggunakan Python dan pustaka SymPy, kita dapat menghitung integral tentu ini:

```python
# Mengimpor pustaka SymPy
import sympy as sp

# Membuat simbol variabel
x = sp.symbols('x')

# Menentukan batas-batas integral
a = 1
b = 5

# Menentukan fungsi
g = 2*x

# Menghitung integral tentu
definite_integral = sp.integrate(g, (x, a, b))

# Menampilkan hasil
print("Integral tentu dari g(x) dari {} hingga {} adalah:".format(a, b), definite_integral)
```
